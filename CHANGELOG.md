## [2.0.0] - 2026-07-15

### ✨ Added

#### 双语翻译系统
- **三模式翻译切换**：中文 / 双语 / 英文 三种显示模式，单按钮循环切换
- **Google 翻译 API 集成**：后端自动调用 Google 翻译 API 生成中文标题
- **翻译缓存机制**：已翻译标题持久化到 `title-zh-cache.json`，避免重复翻译
- **双语字段生成**：每条新闻生成 `title_zh`、`title_en`、`title_bilingual` 三个字段
- **前端响应式切换**：基于 Vue3 响应式系统，切换模式后所有标题即时更新

#### 收藏与历史系统
- **收藏功能**：点击星标收藏/取消收藏新闻，数据存储在 localStorage
- **新增收藏计数**：实时显示新增收藏数量，点击查看后自动清零
- **阅读历史**：自动记录已读新闻，支持最大 100 条历史记录
- **新增历史计数**：实时显示新增已读数量，点击查看后自动清零
- **数据导入导出**：支持收藏和偏好设置的 JSON 格式导入导出
- **单条删除与清空**：收藏/历史列表支持单条删除和一键清空

#### 滚动进度条增强
- **点击跳转**：点击进度条任意位置，页面平滑滚动到对应位置
- **拖拽跳转**：按住进度条拖拽，页面实时跟随滚动
- **百分比显示**：进度条中间显示当前滚动百分比
- **顶部/底部快捷按钮**：一键回到顶部或跳到底部
- **智能显隐**：滚动超过 100px 时显示，顶部时隐藏

#### 日期校验系统
- **未来日期检测**：自动检测发布时间是否超过当前时间 1 小时
- **日期自动修正**：检测到未来日期时，自动替换为抓取时间
- **归档数据修复**：对历史归档数据中的未来日期进行批量修正
- **容错机制**：支持 1 小时宽容度，避免时区差异导致的误判

### 🎨 UI/UX 优化

- **翻译按钮改造**：从三个独立按钮改为单按钮循环切换，布局更对称
- **Emoji 图标**：翻译按钮使用 🇨🇳 / 🌐 / 🇺🇸 图标，适配明暗模式
- **骨架屏加载**：数据加载时显示骨架屏，提升感知体验
- **徽章计数**：收藏和历史按钮显示新增数量徽章
- **过渡动画**：进度条、弹窗等添加平滑过渡动画

### 🔧 后端增强

- **抓取管道优化**：数据采集 → 清洗 → 过滤 → 去重 → 翻译 → 输出 全流程标准化
- **并发控制**：使用 p-limit 限制并发抓取数，避免请求过快
- **多源合并**：支持 13+ 平台、160+ 信息源的数据聚合
- **AI 过滤**：基于关键词的 AI 相关性过滤
- **双时间窗口**：同时生成 24 小时和 7 天两个数据集
- **OPML 支持**：支持从 OPML 文件导入 RSS 订阅源

### 📚 文档体系

- **PRD 文档**：产品需求文档 v2.0，完整描述功能规格
- **ARCH 文档**：架构设计文档，前后端技术栈与系统设计
- **SDD 文档**：详细设计文档，20 个模块的接口与算法设计
- **TEST 文档**：测试用例文档，单元/组件/集成/性能/兼容性测试
- **DEPLOY 文档**：部署文档，Vercel/GitHub Actions/Nginx 部署指南

### 🔄 Changed

- **翻译按钮交互**：从三按钮并列改为单按钮循环切换（zh → bilingual → en）
- **数据结构升级**：新闻条目新增 `title_zh`、`title_en`、`title_bilingual` 字段
- **存储结构升级**：用户偏好新增 `lastSeenFavoritesCount`、`lastSeenHistoryCount`、`translateMode` 字段
- **抓取流程重构**：在去重后、输出前新增日期校验和翻译增强步骤

### 🐛 Fixed

- **未来日期显示错误**：修复 NewsNow 等源发布日期解析为未来时间的问题
- **黑暗模式文字不可见**：翻译按钮改用 Emoji 图标，确保明暗模式下均可见
- **翻译模式切换无响应**：建立完整的双语字段体系，切换模式后标题正确更新
- **日期时间轴不一致**：日期校验确保时间轴日期与实际发布日期一致

### 📦 Dependencies

- `dayjs`：日期处理
- `fast-xml-parser`：XML/RSS 解析
- `opmlparser`：OPML 文件解析
- `p-limit`：并发控制
- `tsx`：TypeScript 执行
- `vue`：^3.4.0 前端框架
- `vite`：构建工具
- `tailwindcss`：CSS 框架
- `@vitejs/plugin-vue`：Vue 3 Vite 插件
- `vite-plugin-pwa`：PWA 支持

## 版本语义化说明

### 版本号格式

```
MAJOR.MINOR.PATCH
```

- **MAJOR**：不兼容的 API 变更或重大功能重构
- **MINOR**：向后兼容的功能性新增
- **PATCH**：向后兼容的问题修正

### 变更类型说明

| 类型 | 说明 |
|:-----|:-----|
| `Added` | 新功能 |
| `Changed` | 对现有功能的变更 |
| `Deprecated` | 即将移除的功能 |
| `Removed` | 已移除的功能 |
| `Fixed` | Bug 修复 |
| `Security` | 安全相关修复 |
| `Performance` | 性能优化 |

4. **描述具体**：每条变更描述清晰，便于理解
5. **关联 Issue**：如有关联 Issue 或 PR，添加链接
